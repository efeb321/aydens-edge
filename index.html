How to load any course

Send these to Claude and a fully-tailored plan comes back:

  • StrackaLine map pages (aerial + green) for each hole — the carries and hazards live there.
  • Back-tee total + tee offset per hole (offset = how far forward Ayden's tee is, in yards; 0 = back tee).
  • Latest Clippd profile if his numbers move (carry, dispersion, SG by category).

The engine computes carries, club, aim and the never-short-side rule from that data. The live wind tool travels with every course.
